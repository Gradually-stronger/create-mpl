# create-mpl

Create a project in seconds!

## Usage

```bash
# getting started
npm init mpl

# or
# scaffold: vite, cra, umi, vue, svelte, angular, github
npm init mpl my-app [scaffold]

# help
npm init mpl -h

# version
npm init mpl -v
```

## Scaffold

- `Vite` - vite
- `React` - cra
- `UmiJS` - umi
- `Vue.js` - vue
- `Svelte` - svelte
- `Angular` - angular
- `GitHub Template (Custom)` - github
